# Contao IsotopeLNbits Bundle



## Getting started

```
composer require digiwerft/isotope-lnbits
```

## Compability

| Contao Version      | PHP Version |
|---------------------|-------------|
| \>= 4.13 \|\| \>= 5.3 | ^8.2        |

## Important note

You need an LNbits Server to use this extension.
If you dont have one, try opago-pay.com.
